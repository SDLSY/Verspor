# 评委增强理解包

本目录是面向评委或外部阅读者的增强理解材料包，用于帮助快速理解项目定位、系统结构、核心页面、流程逻辑、测试可信度和演示方法。

## 阅读入口

- 首先阅读 `00_总索引/`
- 然后按需要进入：
  - `01_正式提交文档/`
  - `02_项目总览与系统理解/`
  - `03_架构与数据库设计/`
  - `04_核心页面与交互原型/`
  - `05_流程逻辑图/`
  - `06_测试与可信度证明/`
  - `07_演示与答辩辅助/`

## 使用规则

- 这是对外解释层，不是源码事实第一来源。
- 若其中内容与当前工程状态冲突，应回到 `docs/00_入口与维护/02_当前项目现状与上下文基线.md` 和对应事实文档重新核对。
