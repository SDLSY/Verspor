# 最终材料目录

本目录用于保存当前对外交付的最终导出件。为避免后续再次整理，交付层按“根目录直接交付件 + 子目录补充件”维护。

## 根目录保留内容

- 正式报告 `docx` / `pdf`
- 演示录屏与产品引入视频
- 软件介绍相关 `pdf`
- 当前交付版协议文档

## 子目录说明

- `安装包/`
  - Android 安装包，例如 `app-release.apk`
- `压缩归档/`
  - 对外分发或备份用的 ZIP 包
- `参考快照/`
  - 交付快照，例如 `README.pdf`
- `协议与附件/`
  - 补充协议附件和历史归档版本

## 协议文件说明

- 根目录 `Hi90B智能指环_通讯协议_V102_20250310.docx` 是当前交付层保留的版本。
- `协议与附件/Hi90B智能指环_通讯协议_V102_20250310_历史归档版.docx` 是从根目录历史材料归档过来的旧版本，用于追溯，不作为默认引用版本。

## 使用规则

- 这里只放最终导出结果，不放调试截图、原始日志或中间稿。
- 若某份材料仍在反复编辑，应先留在 `docs/` 或其他工作目录，确认成为最终交付版后再放入本目录。
- 若新增 APK、ZIP、PDF 快照或协议附件，应优先进入对应子目录，不再回到仓库根目录。
