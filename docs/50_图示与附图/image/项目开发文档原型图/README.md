# 项目开发文档原型图

本目录用于存放按统一版式生成的《项目开发文档》原型图。

统一规则：

- 手机端页面：横版画布，左侧手机框，右侧图注区
- 后台页面：网页后台框，左侧菜单栏 + 顶部栏 + 主内容区
- 手机端保留 5 个一级导航位
- 编号按从上到下、从左到右放置
- 主组件尽量卡片化
