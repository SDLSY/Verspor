# 历史交付与模板

本目录用于收口不再作为当前事实第一来源、但仍需要保留的文档材料。

## 目录结构

- `01_模板/`
  - 各类报告模板与模版 Markdown / Word 文件
- `02_完成版/`
  - 历史完成版 Markdown / Word 文档
- `03_原始导出与提交稿/`
  - 原始 docx 导出稿、提交稿
- `04_过程稿与提纲/`
  - 过程说明、提纲、阶段总结、旧整理稿

## 使用规则

- 这些材料默认不作为当前项目事实的第一来源。
- 若内容与当前代码、当前入口文档冲突，以当前代码与活跃事实文档为准。
- 若需要复用模板或追溯历史提交过程，可从本目录查找，但引用前应回到：
  - `docs/00_入口与维护/02_当前项目现状与上下文基线.md`
  - `docs/00_入口与维护/03_当前材料总索引与归档建议.md`
  - `docs/README.md`

## 维护说明

- 新增模板、完成版、旧提交稿、比赛材料过程稿时，优先放入本目录，而不是继续堆在 `docs/` 顶层。
- 若某份历史材料重新成为当前正式交付物，应从本目录提升到对应的活跃目录，而不是保留双份入口。
