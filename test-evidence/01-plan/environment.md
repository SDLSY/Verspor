# 1.3 测试环境

本轮提交版测试在真实 Android 设备与云端环境均已正确登录、正确联网、正确满足外设前提的条件下完成。环境组件、软件栈与调试工具均处于可用状态，能够支撑功能闭环、模型性能与系统联动测试的稳定执行。

| 环境组件 | 详细配置 | 在测试中的作用 |
| --- | --- | --- |
| Android 真机 | 真实设备，运行 `:app-shell` 调试包 | 承载页面、设备连接、问诊、报告、干预与机器人功能复测 |
| 云端服务 | `cloud-next` 与相关云端能力 | 提供睡眠分析、建议生成、报告理解、讲解文案与 TTS 服务 |
| 本地数据库 | Room 数据库与导出链路 | 验证页面消费、写回、历史记录与状态同步 |
| 调试工具 | adb、logcat、gradle、数据库查看与截图工具 | 用于执行、取证与复核 |
